This is the project about Markov-sentence-generator

Markovian-tweets, due September 26th. Consume a text file with sentences and produce a tweet (< 145 characters) with the style of the input text. I see several layers of success in this project and encourage you to shoot for the level that best matches your team's abilities.
If you do not know how to use it, you can see the 'README' in the 'View On Github' which includes how to use it and introduction about it.
